Задание 1 (Task_01)

Напишите программу, которая знакомится с пользователем и собирает о нём информацию. Сначала программа должна спросить имя, затем возраст и любимое число. Все значения вводятся через input. После этого выведите информацию о пользователе в одной строке с помощью f-строки.

Далее выведите тип каждой введённой переменной с помощью type. Обратите внимание, что возраст и число изначально будут строками. Попробуйте преобразовать возраст в int и снова выведите его тип.

После этого измените значение одной переменной несколько раз. Например, сначала пусть переменная содержит число, потом строку, потом булево значение. После каждого изменения выводите тип.

В конце программы создайте многострочную строку с информацией о пользователе и выведите её. Добавьте также raw-строку с каким-нибудь путём к файлу.
